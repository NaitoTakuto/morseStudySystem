GitHubのテストも兼ねて、
モールス信号体験学習システムのソースコードを公開致します。

python3とGPIOの学習に役立てば幸いです。

 動作環境につきましては、
python3と、「wiringpi」ライブラリに対応した、
Raspberry Pi 3 Model Bでの使用を想定しています。
 同フォルダ内に、回路図、ブレットボード図、
作成ソフト「fritzing」のファイルも同梱しています。


推奨の実行方法。
１．動作環境を満たしたRaspberryPiを用意して下さい。
２．同フォルダ内の図の通りに回路を実装して下さい。
３.main.pyを実行して下さい。
